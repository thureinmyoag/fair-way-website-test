WEB
DESIGN

Course

Photoshop ကို အသုံးပြုပြီး Web Template များကို ဖန်တီးချင်သူများ၊ မိမိကိုယ်တိုင်ဖန်တီးထားသော Web Template များကို HTML, CSS, JavaScript, jQuery အသုံးပြုကာ Web Browser များတွင် မည်ကဲ့သို့ အလုပ်လုပ်သည်ကို လေ့လာချင်သူများ၊ Web Designer နှင့် Front-End Developer ဖြစ်ချင်သူများ၊ တက်သင့်သော သင်တန်းဖြစ်သည်။

Photoshop ကို အသုံးပြုပြီး Web Template များဖန်တီးတည်ဆောက်ပုံနှင့် UI/UX သဘောတရားများကို အခြေခံမှစပြီး သင်ကြားပေးသော သင်တန်းဖြစ်သည်။ HTML, CSS element များကိုု အခြေခံမှ စတင်သင်ကြားပေးမှာဖြစ်ပြီး CSS framework တစ်ခုုဖြစ်သော Bootstrap ကို အသုံ;ပြုကာ Responsive website တစ်ခုကို တည်ဆောက်နိုင်သည်အထိ သင်ကြားပေးသောသင်တန်းဖြစ်သည်။ သင်တန်းကာလအတွင်းတွင်လည်း UI Template များ၊ Website Template Design များကို ဆရာနှင့်အတူ ဖန်တီးသွားမည်ဖြစ်သည်။

လက်တွေ့အသားပေး သင်တန်းဖြစ်ပြီး သင်တန်းပြီးဆုံးပါက Professional Web Designer & Front-End Developer တစ်ဦးအဖြစ် ရပ်တည်ရန် လိုအပ်သည့် ကျွမ်းကျင်မှုများကို ရရှိသွားမည် ဖြစ်သည်။
